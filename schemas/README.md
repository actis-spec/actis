# ACTIS Schemas

Normative JSON schemas for ACTIS v1.0.

| Schema | Description |
|--------|-------------|
| `actis_transcript_v1.json` | Transcript format (actis-transcript/1.0). Required for transcript validation. |
| `actis_manifest_v1.json` | Bundle manifest (manifest.json). Defines core_files and optional_files for evidence bundles. |

Schema identifiers:

- Transcript: `actis-transcript/1.0` (see `$id` in actis_transcript_v1.json)
- Manifest: `actis-manifest/1.0` (see `$id` in actis_manifest_v1.json)
