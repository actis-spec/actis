ACTIS Patent Non-Assert Commitment

Effective Date: 2026-03-08

Non-Assert Covenant

The ACTIS Maintainers irrevocably covenant not to assert any patent claims necessary to implement the ACTIS v1.x specification against any party implementing the specification in good faith.

Scope

This commitment applies to:

  • The ACTIS v1.x specification (as published in the actis/ normative documents)
  • Implementations that conform to the specification

Termination

This non-assert commitment terminates only if a party asserts a patent claim against an ACTIS implementation specifically for implementing ACTIS.
